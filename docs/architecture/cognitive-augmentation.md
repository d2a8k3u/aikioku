
# Layer 6: Cognitive Augmentation

Features that enhance human cognition and learning.

## Spaced Repetition

Anki-compatible SM-2 algorithm for knowledge retention:

- **Auto-generation:** LLM creates flashcards from notes
- **Scheduling:** SM-2 algorithm (again/hard/good/easy)
- **Card types:** Factual, conceptual, application
- **Review UI:** Dedicated review dashboard

### Card Model

```python
class Card:
    id: str
    type: CardType        # factual, conceptual, application
    front: str            # Question
    back: str             # Answer
    source_note_id: str
    status: CardStatus    # new, learning, review, suspended
    interval: int         # Days until next review
    ease_factor: float    # SM-2 ease factor
    due_at: datetime
```

## Progressive Summarization

Multi-level summaries generated by LLM:

| Level | Description | Length |
|-------|-------------|--------|
| 1     | One-line summary | ~1 sentence |
| 2     | Brief summary | ~1 paragraph |
| 3     | Detailed summary | ~3 paragraphs |
| 4     | Full extraction | Key points + structure |

## Serendipity Walks

Random graph traversal for discovery:

1. Start from a given entity
2. Random walk through relations
3. Return discovered entities and their connections
4. Configurable step count

## Cognitive State

Adapts to user context:

- Tracks active project/focus area
- Adjusts retrieval weighting based on context
- Surfaces relevant memories for current task
